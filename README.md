# Run Steps
1. Clone the repository
2. Use Visual Studio 2022 to open the Task2/New/Client-Management-API/Client-Management-API solution
3. Run the solution
4. Open the Task2/New/Client-Management-Frontend folder. I used visual studio code.
5. Run `npm install` to ensure all the packages are installed.
6. Run `npm run dev` to run the frontend application.
7. Access the website via the provided local URL. For me this is 'http://localhost:5173'
